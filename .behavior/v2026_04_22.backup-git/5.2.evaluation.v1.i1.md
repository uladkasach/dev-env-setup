## summary (as implemented)

implemented `git backup` command that:
- parses `--repo all` and `--into s3://...` arguments
- validates all required arguments with clear error messages
- displays tree status before backup via `_git_tree_status --repo @all`
- creates `git.tar.gz` from `~/git/` with `node_modules/` and `.cache/` excluded
- creates `.claude.tar.gz` from `~/.claude/` if present (warns if absent)
- uploads archives to s3 at `$prefix/effectiveAt=$timestamp/~/`
- displays 🍯 confirmation with S3 URIs and file sizes
