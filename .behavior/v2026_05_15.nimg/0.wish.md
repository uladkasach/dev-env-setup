# pqiv with vim keybindings

lightweight image viewer with vim-style navigation.

## install

```sh
sudo apt install pqiv
```

## config

create `~/.pqivrc`:

```ini
[options]
zoom-level=1.0
disable-scaling=1

[keybindings]
h { shift_x(50) }
j { shift_y(-50) }
k { shift_y(50) }
l { shift_x(-50) }
gg { set_shift_align_corner(N) }
G { set_shift_align_corner(S) }
<Control>j { set_scale_level_relative(0.9) }
<Control>k { set_scale_level_relative(1.1) }
```

## keybindings

| key | action |
|-----|--------|
| h | pan left |
| j | pan down |
| k | pan up |
| l | pan right |
| gg | jump to top |
| G | jump to bottom |
| ctrl+k | zoom in |
| ctrl+j | zoom out |
| q | quit |

## usage

```sh
pqiv image.png
pqiv *.png              # browse multiple images
pqiv-wayland image.png  # if on wayland
```

## notes

- opens at 100% zoom by default
- no auto-center on startup (limitation)
- gg/G jump to top/bottom center of image
